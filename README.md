# TanTracker3000 Pro

Commercial GPS tracking platform. Stealth hardwired vehicle tracker with silent listen mode, 60-day location history, and automated billing.

## Files

| File | Purpose |
|---|---|
| `server.js` | Node.js backend — WebSocket relay, Stripe webhooks, admin API, Turso DB |
| `package.json` | Dependencies |
| `device-v1.html` | Production device app (Fully Kiosk) — point customer units here |
| `device-v2.html` | Testing device app — test new features here first |
| `device.html` | Development reference |
| `control.html` | Customer control panel |
| `admin.html` | Admin dashboard (private) |
| `welcome.html` | Post-payment landing page |

## Device Token

Each customer vehicle gets a unique token generated by the server when Stripe confirms payment.
Set the Fully Kiosk start URL to:
```
https://YOUR_GITHUB_PAGES_URL/device-v1.html?token=CUSTOMER_TOKEN
```

## Environment Variables (Render)

```
TURSO_URL               = libsql://your-db.turso.io
TURSO_TOKEN             = your-turso-token
STRIPE_SECRET_KEY       = sk_live_xxx
STRIPE_WEBHOOK_SECRET   = whsec_xxx
STRIPE_INSTALL_PRICE_ID = price_xxx
STRIPE_SINGLE_PRICE_ID  = price_xxx
STRIPE_MULTI_PRICE_ID   = price_xxx
ADMIN_KEY               = your-strong-admin-password
PORT                    = 8080
```

## Business Model

- $99 install fee ($18 to installer, $81 to company)
- $22/month single tracker
- $19/month per tracker for 2+ trackers
- $5/month per active customer to installer (automatic via Stripe Connect)
